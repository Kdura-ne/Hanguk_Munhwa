# 한국 문화
## Trabalho da matéria de PWII e Participação no concurso de vídeos do CECSP

Desenvolvi um site que requer uma página de login, uma página de inscrição, uma página explicando o tema selecionado e, finalmente, um questionário.

Decidi criar o site com um tema cultural coreano para combinar os conhecimentos dos dois cursos que estou a frequentar atualmente. Também pretendo usar este projeto para participar num concurso.

Este site compreende, além do índice, seis páginas sobre temas culturais coreanos do meu interesse, páginas funcionais de login e registo, uma página onde os utilizadores podem responder a um breve questionário sobre os temas abordados e, finalmente, uma página para visualizar os resultados.

O site está atualmente em estado provisório, mas passará por atualizações futuras (para garantir que os utilizadores só possam responder ao questionário uma vez e para incorporar conteúdo adicional à medida que estiver disponível).

(3 de novembro de 2025)

## PWII 과목 과제 및 CECSP 동영상 공모전 참가

로그인 페이지, 회원가입 페이지, 선택한 주제를 설명하는 페이지, 그리고 마지막으로 퀴즈가 필요한 웹사이트를 기획했습니다.

현재 수강 중인 두 과목의 지식을 결합할 수 있도록 한국 문화 테마로 사이트를 제작하기로 결정했습니다. 또한 이 프로젝트를 활용해 공모전에 참가할 예정입니다.

이 사이트는 인덱스 외에도 제가 관심을 갖는 한국 문화 주제 6개 페이지, 기능적인 로그인 및 회원가입 페이지, 다루어진 주제에 대한 간단한 퀴즈를 풀 수 있는 페이지, 그리고 마지막으로 결과를 확인할 수 있는 페이지로 구성되어 있습니다.

사이트는 현재 임시로 완성된 상태이지만 향후 업데이트가 있을 예정입니다(사용자가 퀴즈를 한 번만 풀 수 있도록 하고, 추가할 내용이 생기면 업데이트할 계획입니다).

(2025년 11월 3일)
